# Web Pool D02 - HTML et CSS

## Description

Ce depot contient les exercices du jour 2 de la Web Pool Epitech. Il introduit la mise en page CSS en complement des pages HTML.

## Objectif

L'objectif est de pratiquer la separation entre structure HTML et style CSS, ainsi que les bases du responsive design.

## Stack / technologies

- HTML
- CSS

## Structure

```text
.
└── day02/
    ├── task01/
    ├── task02/
    └── task03/
```

## Fonctionnalites principales

- Pages HTML associees a des fichiers CSS.
- Exercices de couleurs et de blocs.
- Mise en page responsive avec sections, articles, header et footer.

## Lancement

Ouvrir un exercice dans le navigateur :

```bash
open day02/task03/index.html
```

## Remarques

Chaque dossier `taskXX` correspond a un exercice independant.
